# MÆSTRO

MÆSTRO（マエストロ）はシンプルな合成パズルゲームです。
画面上に散らばる「媒体」同士をドラッグで重ねてあたらしい「媒体」を「錬成」してください。「錬成」を繰り返すことで、どんどんと新しい「媒体」を生み出すことができます。どの組み合わせで何ができるかは、試してみるまでわかりません。
2週間の期間内で「錬成」を繰り返して、より高いスコアを目指してください。
